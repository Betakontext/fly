This is a little fly betweeen you and your website.
Catch, fork and have fun.

This project is build with AI assistance and deployed under MIT licence.

To run locally download the whole repository and start a local http-server inside the root directory,
f.e. bash: python3 -m http.server 8000
Then open http://localhost:8000/ in your browser.
ctrl+c in the terminal closes the server.
