This is a fly betweeen you and your website.
Catch and fork it for further fun developments.

Online version: https://dev.betakontext.de

To run it locally download the whole repository and start a local http-server inside the root directory,
f.e. bash: python3 -m http.server 8000
Then open http://localhost:8000/ in your browser.
ctrl+c in the terminal closes the server.
