There is enough space betweeen screen and your content for
a little fly to be stuck in between.

For online version see: https://dev.betakontext.de
For local testing download the whole repository and start a local http-server inside the root directory,
f.e. bash: python3 -m http.server 8000
Then open http://localhost:8000/ in your browser.
ctrl+c in the terminal closes the server.

Catch it, fork it, have fun.
